# Invite App Homework

## Compiling

To create fat jar:

```bash
sbt assembly
```

## Running

This application can bu run with the command:

```bash
sbt run
```

This is the same as running:

```bash
./run.sh
```

Use fat jar. if the fat jar is created then:

```bash
java -jar target/scala-2.12/invite-homework.jar
```
